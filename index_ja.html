1.Welcome To TUBAME


1.1.TUBAMEとは


TUBAMEはJavaアプリケーションのマイグレーション時に必要となる、修正箇所の算出や修正工数の算出を自動で行うためのツールです。自動化により作業工数の削減や人的ミスの防止が可能となり、併せてマイグレーションに関わるノウハウを提供しているため作業精度の向上にも役立ちます。

複数のサブツールを備えており、Eclipseプラグインとして利用することができます。OSS製品としてTUBAMEコミュニティで公開されておりますので是非ご活用ください。

TUBAMEはJavaアプリケーションのスムーズな移植を支援するために用意されたツールで、以下3つのツールから構成されています。

  1. ナレッジベース検索ツール(tubame.portability)

    • TUBAMEコミュニティで管理しているナレッジベースより検索します

    • 修正項目毎に移植難易度や修正工数を表します

    • 修正項目毎に修正ガイドラインを参照することで修正方法が確認できます

  2. 依存性検索ツール(tubame.wsearch)

    • 移行先の環境(ミドルウェア)で利用できないJavaパッケージやクラス情報を自動で抽出します

    • 移行先の環境として、ライブラリ情報を別途設定することで、フレームワークについても、移行先の環境で利用できないAPI情報を抽出します

  3. ナレッジ管理ツール(tubame.knowhow)

    • ナレッジベース検索に必要なナレッジを登録及び編集できます

    • ナレッジコンテンツの作成から、検索条件の設定まで、ナレッジベース検索で必要な情報をこのツールだけで設定できます

Note

  • なぜTUBAME?

    日本では、燕が家に巣を作るとその家に幸福をもたらすという言い伝えがあります。このツールを利用してあなたのJBossマイグレーションにも幸福が訪れることを祈ります！



1.2.ツール概要


  1. ナレッジベース検索ツール

    Figure 1. ナレッジベース検索ツールの入力と出力について


    Note

    • マイグレーションナレッジXML

      ナレッジベース検索ツール使用時には必須のファイルとなります。移行ナレッジはTUBAMEコミュニティからダウンロード、または、ナレッジ管理ツールで自ら作成することもできます。

    • マイグレーションガイド（修正ガイドライン）

      ナレッジベース検索ツール使用時には任意のファイルとなります。検索結果からマイグレーションガイドの参照を行う場合、TUBAMEコミュニティからダウンロードしたマイグレーションガイド（HTML）を任意の場所に配置し、Preference（TUBAME -> ナレッジベース検索ツール -> マイグレーションガイド設定）でマイグレーションガイドのパスを設定してください。
    Note

    現在、コミュニティで公開されているマイグレーションナレッジXML及びマイグレーションガイドは以下よりダウンロード可能です。

    ダウンロードしたファイルを検索対象のプロジェクト配下に配置し、マイグレーションガイドのパス設定を行い、ナレッジベース検索実行時にマイグレーションナレッジXMLを参照することで利用可能です。

    • JBossEAP4->JBossEAP6 （マイグレーションガイドの参照はこちら）

    • WebLogic9->JBossEAP6 （マイグレーションガイドの参照はこちら）

    • WebSphere6->JBossEAP6 （マイグレーションガイドの参照はこちら）

    • Tomcat5.5->Tomcat6.0 （マイグレーションガイドの参照はこちら）

    • Tomcat5.5->Tomcat7.0 （マイグレーションガイドの参照はこちら）

    • Tomcat6.0->Tomcat7.0 （マイグレーションガイドの参照はこちら）

    • Tomcat7.0->Tomcat8.0 （マイグレーションガイドの参照はこちら）

  2. 移植性検索ツール

    Figure 2. 移植性検索ツールの入力と出力について


    Note

    • 移行先環境のライブラリ情報

      移植性検索ツール使用時には必須の情報となります。デフォルトではJBossEAP6が移行先ライブラリとして設定されています。

      移行先ライブラリをデフォルトのJBossEAP6から変更したい、また、フレームワークなどの別のライブラリを追加したい場合、ライブラリ情報をPreference（TUBAME -> 依存性検索ツール -> ライブラリ設定）で設定してください。移行先ライブラリとして設定する情報は以下になります。

      1. 移行先ライブラリのバイナリ（zip）を指定する

      2. 移行先ライブラリを含むMavenリポジトリ（zip）を指定する
